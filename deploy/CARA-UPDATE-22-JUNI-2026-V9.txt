=== LANGKAH UPLOAD (File Manager) ===

1. Login DirectAdmin / cPanel Qwords → File Manager

2. Masuk folder ROOT project:
   chatbot-keperawatan.damgocompany.com/
   (tempat file artisan — BUKAN hanya folder public)

3. Upload ZIP:
   chatbot-keperawatan-update-22juni2026-v9.zip

4. Extract di folder tersebut → pilih OVERWRITE / timpa semua
   (kecuali .env — pastikan .env lama masih ada)

5. HAPUS file public/hot jika muncul (WAJIB — rusak CSS mobile)

6. Copy deploy/clear-cache.php → public/clear-cache.php (jika belum ada)

7. Buka browser:
   https://chatbot-keperawatan.damgocompany.com/clear-cache.php?key=ck2026fix
   Tunggu "SELESAI"

8. HAPUS dari server:
   - public/clear-cache.php
   - file ZIP
   - CARA-UPDATE-22-JUNI-2026-V9.txt (opsional)
